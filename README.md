# Hemmingway Technologies

Professional company website for Hemmingway Technologies Private Limited.

## 📂 Project Structure

```
hemmingway-technologies/
├── company-website/          # Main React application
│   ├── src/
│   │   ├── components/       # Reusable React components
│   │   │   ├── layout/       # Layout components (Navbar, Footer)
│   │   │   ├── ui/           # UI components (Cards, Loaders, Text)
│   │   │   └── index.js      # Component exports
│   │   ├── pages/            # Page components (Home, About, Team, Blog, Contact)
│   │   ├── hooks/            # Custom React hooks
│   │   ├── styles/           # CSS styles
│   │   ├── assets/           # Static assets
│   │   ├── App.jsx           # Main App component
│   │   ├── main.jsx          # Entry point
│   │   └── index.css         # Base styles
│   ├── public/               # Static files
│   │   ├── logos/            # Logo assets and favicons
│   │   ├── _redirects        # Netlify/hosting redirects
│   │   └── *.png, *.svg      # Images and icons
│   ├── package.json          # Dependencies
│   ├── vite.config.js        # Vite configuration
│   └── eslint.config.js      # ESLint configuration
├── README.md                 # This file
└── package-lock.json         # Dependency lock file
```

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation & Development

```bash
# Navigate to the project
cd company-website

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Pages

- **Home** (`/`) - Landing page with hero, services, testimonials
- **About** (`/about`) - Company story, values, mission
- **Team** (`/team`) - Team member profiles and culture
- **Blog** (`/blog`) - Technical blog with filtering
- **Contact** (`/contact`) - Contact form and information

## Tech Stack

- **React 19** - UI framework
- **Vite** - Fast build tool
- **React Router v7** - Client-side routing
- **GSAP** - Animation library
- **Lenis** - Smooth scrolling
- **ESLint** - Code quality
- **GitHub Pages** - Deployment
